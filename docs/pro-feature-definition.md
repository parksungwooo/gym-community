# Gym Community Pro 기능 정의

## 6단계 목표

Pro는 무료 기능의 단순 확장이 아니라, 사용자가 매일 남기는 기록을 다음 행동으로 바꿔주는 유료 코치 경험이어야 한다.

무료 사용자는 “기록하고 경쟁하는 재미”를 얻고, Pro 사용자는 “내 몸에 맞춘 다음 운동, 더 큰 보상, 더 선명한 성장 증거”를 얻는다.

## Pro 핵심 기능

### 1. AI 주간/월간 맞춤 운동 플랜

- Pro로 둬야 하는 이유: 가장 강력한 결제 가치다. 사용자는 기록 앱보다 “내 상황을 읽어주는 코치”에 돈을 낸다.
- 입력 데이터: 체력 테스트 결과, 최근 운동 기록, 주간 목표, 스트릭, 체중/목표 체중, 회복 신호.
- 제공 가치: 오늘 운동, 이번 주 핵심 세션, 회복일, 월간 성장 블록, 점진 과부하 목표.
- 결제 동기: “오늘 뭐 하지?”를 없애고, 운동을 시작하기 쉽게 만든다.

### 2. 상세 운동 분석 + AI 인사이트 대시보드

- Pro로 둬야 하는 이유: 기록이 쌓일수록 분석 가치가 커진다.
- 제공 가치: 볼륨 변화, XP 추이, 스트릭, 체중 변화, 1RM 예측, 회복 추천, 정체 구간 감지.
- 결제 동기: 사용자가 “내가 잘하고 있나?”를 확인하고 싶어지는 순간에 강하게 작동한다.

### 3. Pro 전용 리그

- Pro로 둬야 하는 이유: 경쟁형 사용자는 보상과 지위를 위해 결제한다.
- 제공 가치: 일반 리그보다 1.5배 높은 XP 보상, Pro 챔피언 배지, 전용 상위권 보상.
- 결제 동기: “무료 리그도 재밌지만, Pro 리그는 보상이 더 세다”는 차이가 명확하다.

### 4. 광고 제거 + 무제한 기록

- Pro로 둬야 하는 이유: 기록 개수 제한은 자연스러운 업그레이드 타이밍이다.
- 무료 제한: 운동 기록 30개.
- 제공 가치: 무제한 기록, 광고 없는 피드, 장기 성장 리포트, 공유 카드 누적.
- 결제 동기: 기록이 쌓인 사용자는 데이터를 잃고 싶지 않다.

### 5. Pro 배지 + 파티 전용 혜택

- Pro로 둬야 하는 이유: 소셜 기능은 유지율을 올리는 유료 가치다.
- 제공 가치: Pro 배지, 파티 목표 달성 추가 보상, 친구 초대 보상, 비공개 챌린지 준비.
- 결제 동기: 혼자 쓰는 앱이 아니라 친구와 함께 쓰는 서비스가 된다.

## Gamification 연계

- Pro 리그 보상: 일반 리그 보상의 1.5배.
- Pro 특별 미션: AI 플랜 시작, Pro 리그 상위 10%, 파티 공동 미션 완료.
- Referral 보상: 초대한 친구가 Pro 결제 시 초대한 사용자에게 +300 XP.
- 파티 보상: Pro 파티 목표 달성 시 +120 XP와 전용 배지.
- 레벨/배지 노출: 프로필, 리더보드, 파티 위젯에서 Pro 상태가 보이게 한다.

## Paywall 메시지 원칙

- 기능 나열보다 삶의 변화 중심으로 말한다.
- “분석 가능”보다 “다음 운동이 바로 정해진다”가 더 강하다.
- “무제한”보다 “기록이 끊기지 않는다”가 더 감정적이다.
- “Pro 배지”보다 “꾸준한 사람처럼 보인다”가 더 설득력 있다.
